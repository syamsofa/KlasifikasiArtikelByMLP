         Sleman - Api obor Asian Games 2018 dari India telah tiba di Lanud TNI AU Adisutjipto Yogyakarta. Torch relay siap dimulai  besok. 

Api obor Asian Games tiba di Lanud Adisutjipto, Selasa (17/7/2018) pukul 08.00 WIB. Api itu diinapkan di Museum Dirgantara nanti malam. Kemudian, Rabu besok sore, api dari India itu akan dibawa ke Candi Prambanan untuk dikawinkan dengan api yang diambil dari Mrapen, Grobogan, Jawa Tengah.

"Mulai besok, api dari India ini akan dikawinkan dengan api dari Mrapen (Jawa Tengah) di Prambanan, kemudian dibawa berkeliling menuju 53 kota di Indonesia sejauh 18.000 kilometer," kata Ketua INASGOC atau panitia pelaksana Asian Games 2018, Eric Thohir, saat jumpa pers di Museum Dirgantara TNI AU, Lanud Adisutjipto Yogyakarta, Selasa (17/7/2018).

                      Baca juga: Api Obor Asian Games 2018 Tiba di Yogyakarta

Rabu (18/7) malam, api akan diinapkan di Pagelaran Keraton Yogyakarta. Selanjutnya pada Kamis (19/7), api Asian Games akan dikirab berkeliling Kota Yogya sebelum dibawa menuju Kota Solo. 

Rute kirab di Yogya dimulai dari Pagelaran Keraton-Titik Nol Kilometer ke arah barat-PKU Muhammadiyah ke utara-Jalan Bhayangkara-Pasar Kembang ke timur ke Teteg Malioboro. Dari Teteg kemudian masuk ke Jalan Malioboro-Titik Nol-Jalan Senopati-Perempatan Gondomanan ke selatan-Pojok Beteng Wetan-Pojok Beteng Kulon ke arah utara-Ngabean-Ngampilan ke arah barat-Samsat-perempatan Pingit-Tugu.

Di Tugu Yogyakarta, api akan dilepas oleh Wali Kota Yogyakarta dan diserahkan ke INASGOC untuk dibawa ke Solo.

Kepala Staf TNI AU, Marsekal TNI Yuyu Sutisna menyatakan TNI AU siap mendukung rangkaian torch relay api Asian Games 2018.

"Rangkaian kegiatan di Prambanan, kemudian ke Solo, Malang, Denpasar, Lombok, Sorong, di Sorong kami siapkan pesawat kecil untuk dibawa ke Raja Ampat, kembali lagi ke Sorong, Makassar, Banjarmasin, Aceh, Medan, Pekanbaru, Padang, Jambi, dan berakhir di Jakarta pada 15 Agustus 2018," ujarnya. 

"Semua sudah kami persiapkan baik dukungan personel, pesawat, dan kru pengawal serta pengamanan. Tadi kedatangan pesawat Boeing 737 yang membawa api obor Asian Games dari India, dijemput dan dikawal pesawat tempur TNI AU," dia menjelaskan.
