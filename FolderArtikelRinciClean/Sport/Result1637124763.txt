         Jakarta - Kirab obor Asian Games 2018 dimulai secara resmi besok. Dari Yogyakarta, api berkeliling Indonesia. 

Api obor Asian Games dari India tiba di Lanud Adisutjipto, Selasa (17/7/2018) pukul 08.00 WIB. Api itu akan dibawa ke Candi Prambanan untuk dikawinkan dengan api yang diambil dari Mrapen, Grobogan, Jawa Tengah.

"Suatu kebanggaan dan kehormatan buat saya dipercaya untuk menjadi brand ambassador torch raley Asian Games 2018, khususnya saat membawa api dari India menuju Indonesia, cukup panjang dan menegangkan karena harus melewati prosedur-prosedur dan menjaga agar api itu menyala terus," kata Susy dalam pesan singkatnya kepada detikSport, Selasa (7/7/2018).

                      Kirab obor akan menempuh jarak 18 ribu km, melalui 54 kota dan kabupaten di 18 provinsi, dari Aceh sampai Papua Barat. Api itu akan menjadi sumber api Asian Games di Stadion utama Gelora Bung Karno (SUGBK), Senayan, saat pembukaan. Sehari sebelumnya, api Presiden RI Joko Widodo di Istana Negara pada 17 Agustus.
