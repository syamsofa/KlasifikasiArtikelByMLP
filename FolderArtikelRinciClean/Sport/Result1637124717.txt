Jakarta - Kirab obor Asian Para Games (APG) 2018 dimulai setelah api diambil dari Api Abadi Mrapen, Desa Manggarmas, Kecamatan Godong, Kabupaten Grobogan, Rabu (5/9). Api dibawa dalam lentera khusus yang akan diantarkan ke Solo.

Hadir dalam kirab obor itu Bupati Grobogan Sri Sumarni hingga Sekjen Asian Para Games 2018 Organizing Committee (INAPGOC), Bayu Rahardian, serta sjeumlah atlet difabel. Acara dimeriahkan tari Gambyong, kemudian penyulutan api dari Api Abadi Mrapen, oleh perwakilan Kraton Solo Kanjeng Mas Ayu Arum Ajeng Mas Kenyowursito. 

Api ditaruh dalam lentera dan diserahkan ke Bupati Grobogan, Jawa Tengah. Bupati Sri Sumarni menyerahkan kepada Bayu untuk dibawa ke Solo menggunakan mobil jip.

                      Bayu, dalam sambutan, mengatakan Indonesia bisa sukses penyelenggaraan dan prestasi. Untuk prestasi, Indonesia memasang target finis pada urutan kedelapan.

"Olahraga menyatukan semua. Menjunjung sportivitas," kata Bayu di lokasi.

"Kobaran api impian, kobaran semangat, yang nantinya menyala terus," ujarnya.

Bayu sekaligus mengingatkan Asian Para Games bukan semata-mata bersaing untuk meraih medali. Tapi sekaligus menjunjung kemampuan disabilitas. 

"Menyuarakan paralimpic, atau paralimpic movement," dia menambahkan. 

Sementara itu, Bupati Grobogan, Sri Sumarni, mengatakan api yang diambil dari Mrapen itu bakal dibawa berkeliling ke sejumlah kota. Hal itu merupakan kebanggaan tersendiri bagi warga Grobogan. 

"Ini jadi kebanggaan kami. Semoga ini tidak kalah kemeriahannya dengan Asian Games," kata Sri dalam sambutannya.

 Lentera lalu langsung dibawa dengan kendaraan menuju kantor National Paralympic Committee (NPC) di Jalan Ir Sutami, Solo.
