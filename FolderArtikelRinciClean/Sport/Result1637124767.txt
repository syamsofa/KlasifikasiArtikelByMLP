         Yogyakarta - Kirab obor Asian Games 2018 bakal dimulai dari Yogyakarta. Api Asian Games tiba dari India besok. 

Api abadi dari India untuk obor akan mendarat di Yogyakarta pada hari Selasa 17 Juli 2018 di bandara Adisutjipto, Yogyakarta. Obor Asian Games akan diinapkan di Yogyakarta selama tiga hari.

"Tanggal 17 besok, api obor Asian Games yang diambil dari India akan mendarat di Yogyakarta langsung dari India, diperkirakan tiba jam 08.00 pagi," kata Asisten Pemerintahan dan Kesejahteraan Rakyat (Kesra) Sekretariat Daerah (Setda) Daerah Istimewa Yogyakarta (DIY), Sulistyo, di kantor Gubernur DIY, Senin (16/7/2018).

Sampai di Yogyakarta, api obor Asian Games ini akan diterima oleh Gubernur DIY. Api kemudian diinapkan di museum Dirgantara di Adisutjipto. Sebelum dibawa ke Prambanan, api terlebih dahulu mampir ke candi Boko. Kemudian, pada tanggal 18 Juli akan dilakukan pengambilan api abadi dari Mrapen, Purwodadi, Jawa Tengah yang akan dibawa ke Prambanan melalui Semarang.

"Kemudian bertemulah di Prambanan pada malam harinya (tanggal 18 Juli) akan disatukan dengan api yang dari India oleh Wapres di Candi Prambanan," ujar Sulistyo.

Setelah disatukan, api akan diinapkan di pagelaran Kraton Yogyakarta. Kemuidan pagi harinya, tanggal 19 Juli api yang sudah disatukan itu akan dikirab keliling Yogyakarta. Rute kirab dimulai dari Pagelaran Kraton-Tititk Nol Kilometer kearah barat-PKU Muhammadiyah ke utara-Jalan Bhayangkara-Pasar Kembang ke timur ke Teteg Malioboro. Dari Teteg kemudian masuk ke jalan Malioboro-Titik Nol-jalan Senopati-Perempatan Gondomanan ke selatan-Pojok Beteng Wetan-Pojok Beteng Kulon (barat) ke arah utara-Ngabean-Ngampilan ke arah barat-Samsat-perempatan Pingit-Tugu. Di Tugu, api akan dilepas oelh Walikota Yogyakarta diserahkan ke Inasgoc untuk dibawa ke Solo. 

"Baru kali ini ketempatan api Asian Games ini," katanya.

Pada kesempatan yang sama, Virza Rezkyana Indra, sebagai event organizer mengatakan, yang akan datang membawa api dari India adalah Erick Tohir sebagai ketua INASGOC, Susy Susanti, pelari dari Yogya dan perwakilan atlit nasional. Menyambut kedatangan api dari India di Adisutjipto akan digelar atraksi oleh TNI.
