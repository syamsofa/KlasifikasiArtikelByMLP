Yogyakarta -

Yogyakarta akan menjadi tempat start kirab obor Asian Games 2018. Di Yogyakarta pula dua api abadi untuk obor tersebut dikawinkan.

Kirab obor Asian Games akan dimulai Kamis (19/7/2018) besok. Yogyakarta akan menjadi kota pertama dari 54 kota yang dilewati obor ajang multi event terbesar se-Asia tersebut.

Sebelum kirab dimulai, terlebih dulu akan dilakukan upaya penyatuan api abadi. Seperti diketahui, api untuk Asian Games diambil dari dua sumber api abadi yakni di New Delhi, India dan Mrapen, Grobogan, Jawa Tengah.

                      Api dari India sudah lebih dulu di Yogyakarta pada Selasa (17/7) kemarin. Sementara api dari Mrapen saat ini, Rabu (18/7) masih dalam perjalanan dari Semarang menuju Yogyakarta.

Kedua api ini akan dipertemukan di Prambanan pada sore ini untuk dikawinkan dan menjadi api untuk obor Asian Games 2018. Obor Asian Games 2018 lantas diinapkan di Pagelaran Keraton Yogyakarta sebelum kemudian mulai dikirab pada Kamis (19/7) besok menuju Solo.

Untuk rute kirab di Yogyakarta sendiri akan berada di seputaran Keraton dan Malioboro. Kirab dimulai dari Pagelaran Keraton ke Titik Nol Kilometer, lalu menuju ke arah barat dan berbelok ke utara di RS PKU Muhammadiyah.

Arak-arakan lalu melewati Jl Bhayangkara hingga mencapai ke Pasar Kembang, untuk kemudian berbelok ke timur ke arah Teteg Malioboro. Lalu obor Asian Games akan melewati jalan Malioboro, kembali ke Titik Nol Kilometer, dan berbelok ke timur ke arah perempatan Gondomanan.

Selanjutnya perjalanan mengarah ke Selatan sampai ke Pojok Beteng Wetan, belok ke barat sampai Pojok Beteng Kulon, lalu ke utara sampai ke Ngabean, berturut-turut kemudian ke Ngampilan, Samsat Yogyakarta, perempatan Pingit, lalu ke Tugu.

Di Tugu, obor Asian Games dilepas Wali Kota Yogyakarta dan diserahkan ke INASGOC untuk melanjutkan perjalanan ke Solo.

Setelah Solo, berturut-turut obor Asian Games 2018 bakal dibawa di antaranya ke Malang, Denpasar, Lombok, Sorong, Raja Ampat, Makassar, Banjarmasin, Aceh, Medan, Pekanbaru, Padang, Jambi, dan Jakarta.
