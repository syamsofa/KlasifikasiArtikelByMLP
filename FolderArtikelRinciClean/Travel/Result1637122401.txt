Nepal Van Java ditutup karena Pemberlakuan Pembatasan Kegiatan Masyarakat (PPKM) level 3 virus Corona. Sebuah kesempatan untuk menambah fasilitas buat wisatawan.

Destinasi wisata Nepal Van Java berada di Dusun Butuh, Desa Temanggung, Kecamatan Kaliangkrik, Kabupaten Magelang, Jawa Tengah. Lokasi ini persisnya berada di lereng Gunung Sumbing.Baca juga: Magelang Masuk Zona Merah, Nepal Van Java Tutup Sementara


 
Semenjak wilayah Kabupaten Magelang masuk ke dalam zona merah Covid-19, seluruh destinasi wisata ditutup sementara. Termasuk, Nepal Van Java.

Kemudian diterapkan PPKM Darurat dan sekarang dilanjutkan PPKM level 3 di Kabupaten Magelang, destinasi wisata Nepal Van Java juga masih ditutup sementara.
 
Kepala Dusun Butuh, Lilik Setiyawan, mengatakan penutupan saat PPKM level 3 ini dimanfaatkan untuk melakukan pembenahan dan penertiban fasilitas yang ada. Di antaranya, penambahan fasilitas tempat sampah dan pengembangan taman.
 
"Kami manfaatkan (penutupan karena PPKM) untuk pembenahan dan penertiban fasilitas yang ada. Seperti penambahan fasilitas tempat sampah, pengembangan di Taman Depok yaitu pemasangan Gunungan Wayang dan tulisan aksara Jawa Dusun Butuh dengan tujuan untuk lebih menonjolkan sisi identitas budaya Jawa," kata Lilik dalam pesan melalui Whatsapp kepada detikTravel, Kamis (29/7/2021).Baca juga: Liburan ke Nepal Van Java, Begini Rute dan Harga Tiketnya


 
Selain itu, keberadaan destinasi wisata Nepal Van Java bukan hanya pemandangan saja. Untuk itu, pengelola menampilkan sisi budaya juga. Nantinya jika situasinya sudah membaik dan diberikan izin akan ditampilkan kesenian yang asli di Dusun Butuh.
 
"Wisata Nepal Van Java Dusun Butuh, tidak hanya pemandangan saja, tapi masih banyak sisi budaya yang bisa kami tampilkan," kata Lilik.

"Sambil menunggu keadaan membaik dan tentunya perizinan kami akan menampilkan juga kesenian yang ada di Dusun Butuh yaitu Dayakan, Kuda lumping, Jathilan, Kubro siswo dan Topeng Ireng sebagai suguhan untuk wisatawan di setiap bulan," dia menambahkan.Baca juga: Menikmati Pesona Nepal di Magelang?
