Pengelola Nepal Van Java akhirnya menjalani vaksinasi Covid-19. Vaksinasi diberikan agar para pengelola terlindungi jika Nepal Van Java dibuka kembali.

Destinasi wisata Nepal Van Java berada di Dusun Butuh, Desa Temanggung, Kecamatan Kaliangkrik, Kabupaten Magelang, Jawa Tengah. Lokasi ini persis berada di lereng Gunung Sumbing.

Adapun vaksinasi tersebut bertempat di basecamp jalur pendakian Gunung Sumbing di Dusun Butuh. Untuk sasaran vaksin ini meliputi crew basecamp pendakian, crew wisata, pedagang, crew ojek gunung dan crew ojek wisata.

Kepala Dusun Butuh, Lilik Setiyawan mengatakan, vaksinasi atas kuota dari Pemkab Magelang untuk Desa Temanggung yang diprioritaskan bagi para pelaku wisata di Dusun Butuh. Hal ini karena Dusun Butuh sangat berhubungan dengan aktivitas wisata agar aman dan terlindungi.

"Vaksinasi dilaksanakan atas dasar kuota yang diberikan Pemkab Magelang melalui puskesmas untuk Desa Temanggung dan diprioritaskan di Dusun Butuh. Tawaran itu kami terima dengan sangat baik karena berhubungan aktivitas wisata warga Dusun Butuh agar aman dan terlindungi," kata Lilik dalam pesan singkat kepada detikTravel, Jumat (6/8/2021).Baca juga: Magelang Masuk Zona Merah, Nepal Van Java Tutup Sementara

Menurut Lilik, sasaran vaksinasi untuk crew wisata, basecamp pendakian, pedagang, ojek gunung dan ojek wisata. Untuk sasaran sebanyak 200, namun yang datang ada sekitar 50-an.

"Sasarannya adalah para crew basecamp pendakian, crew wisata, pedagang wisata, crew ojek gunung dan crew ojek wisata. Dari target 200 orang yang hadir hanya seperempatnya sekitar 50-an orang karena ada beberapa kendala. Selain takut, ada yang sedang sakit jadi mereka tidak hadir dan akan kami ajukan ke tahap vaksinasi selanjutnya," tutupnya.Baca juga: Manfaatkan PPKM, Nepal van Java Berbenah
